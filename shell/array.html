Array

配列

bash 配列まとめ bashの配列変数に関するTips - y-kawazの日記 シェルスクリプトで、ある順番以降の引数を取得する

添字

for文での処理ではbashとzsh番号のずれを意識せずに記述できるが，

ARRAY=("aaa" "bbb" "ccc" "ddd" "eee")
for X in ${ARRAY[*]}; do
  echo ${X}
done

展開

${keyword[@]}:配列を展開 ${@:1}:第一引数以降

local keyword=${@:1} # 配列を文字列として代入
local keyword=(${@:1}) # 配列を配列として代入
